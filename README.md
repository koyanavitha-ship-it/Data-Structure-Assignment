## 📝 Assignment Description
This assignment focuses on the practical implementation and application of two fundamental data structures: **Stack** and **Queue** using the C programming language.

The objective of this assignment is to understand how these data structures work internally and how they can be applied to solve real-world problems efficiently.

The following problems are implemented:

### 🔹 Stack Applications
- Reversing a string using stack operations (LIFO principle)
- Checking for balanced parentheses in an expression
- Finding the Next Greater Element using stack

### 🔹 Queue Applications
- Simulating a printer queue system using basic queue operations (FIFO principle)
- Implementing a Circular Queue with operations like enqueue, dequeue, peek, and display

## 🎯 Learning Outcomes
